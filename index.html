题目

JZ4

输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

思路

https://leetcode-cn.com/problems/zhong-jian-er-cha-shu-lcof/solution/mian-shi-ti-07-zhong-jian-er-cha-shu-di-gui-fa-qin/

  • 递归：
    • 前序：根结点：左子树的前序遍历：右子树的前序遍历
    • 中序：左子树的中序遍历：根结点：右子树的中序遍历
    • 首先根据前序得到第一个根结点，然后在中序中定位到该结点，那么上述的左子树的前序和中序对应的是同一棵树，右子树同理，那么问题变成了根据左右子树的前序和中序遍历结果，找到其根结点并接到最初的根结点的左右位置
    • 这样实际上是一个递归问题，不断通过前序中序结果得到新的根节点，如此递归构造左右子树
    • 注意：因为树中元素不重复，用hashmap存储中序遍历的结果方便定位根结点

空间：O(N)

题目

JZ61

请实现两个函数，分别用来序列化和反序列化二叉树

二叉树的序列化是指：把一棵二叉树按照某种遍历方式的结果以某种格式保存为字符串，从而使得内存中建立起来的二叉树可以持久保存。序列化可以基于先序、中序、后序、层序的二叉树遍历方式来进行修改，序列化的结果是一个字符串，序列化时通过 某种符号表示空节点（#），以 ！ 表示一个结点值的结束（value!）。

二叉树的反序列化是指：根据某种遍历顺序得到的序列化字符串结果str，重构二叉树。

例如，我们可以把一个只有根节点为1的二叉树序列化为”1,”，然后通过自己的函数来解析回这个二叉树

思路

  • BFS
    • 序列化：BFS遍历树，不管node的左右结点是否存在，都要加入队列；结点出队时，如果结点不存在，返回值中加null，存在则加入结点值的字符串形式
    • 反序列化：也利用队列新建二叉树；先将str转换为列表再遍历，不为null则加入二叉树，同时将结点入队；队列为空时遍历完毕，返回根结点
    • https://leetcode-cn.com/problems/xu-lie-hua-er-cha-shu-lcof/solution/jian-zhi-offer-37-xu-lie-hua-er-cha-shu-zgixr/

题目

JZ39

输入一棵二叉树，判断该二叉树是否是平衡二叉树。

在这里，我们只需要考虑其平衡性，不需要考虑其是不是排序二叉树

平衡二叉树（Balanced Binary Tree），具有以下性质：它是一棵空树或它的左右两个子树的高度差的绝对值不超过1，并且左右两个子树都是一棵平衡二叉树。

思路

  • 用求二叉树深度的办法来解决https://leetcode-cn.com/problems/ping-heng-er-cha-shu-lcof/solution/mian-shi-ti-55-ii-ping-heng-er-cha-shu-cong-di-zhi/
  • 后序遍历 + 剪枝 （从底至顶）
    • 思想：后序，从下往上检查子树深度。若平衡，把子树的深度向上传递，不平衡则用-1标记向上传递。这样可以减少重复计算，比如子树早就是非平衡了，它的parent结点不可能是平衡树了，因此直接用-1标记并把它在后面的深度计算中去除
  • 先序遍历 + 判断深度 （从顶至底）
    • 先构造一个计算树深度的函数
    • 根据这个函数从上往下递归判断：当前根节点是平衡树而且它的左右子树都平衡的条件下，根节点才是平衡的

代码

后序遍历 + 剪枝

时间：O(N)

空间：O(N) 递归栈空间

题目

JZ38

输入一棵二叉树，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度

树的题目一定要理解好递归

思路

  • 后序遍历（DFS）递归

    • 此树的深度和其左（右）子树的深度之间的关系。显然，此树的深度 等于 左子树的深度 与 右子树的深度 中的 最大值 +1+1 。
    • 递归的终止条件：root为空，此时返回深度为0
    • 递归：就是对树做后序遍历
      • 计算左子树的深度
      • 计算右子树的深度
    • 返回值：当前树的深度即 左右子树深度的和+1

  • 层序遍历（BFS）：每遍历一层，则计数器 +1+1 。借助队列实现 迭代

    • 特例：root空，深度为0

    • 初始化：队列queue（存储根节点），计数器res

    • 循环体：

      • 初始化一个空列表tmp（存储下一层的各个结点）

      • 遍历队列中的各结点，将其左右子节点加入tmp

      • 更新队列，把tmp元素加入队列

      • 更新层数res

        返回res

题目

JZ57

给定一个二叉树其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的next指针。

思路

  • 中序遍历：根据中序遍历的特性讨论：
    • 有右子树：下一结点应该是右子树的最左结点
    • 无右子树且当前结点使其父结点的左子树：下一结点应是其父结点
    • 无右子树且当前结点是其父结点的右子树：
      • 当前结点的父结点是上一层的左子树：返回父结点的父结点
      • 当前结点的父结点是上一层的右子树，继续向上找，直到某个结点是它父结点的左子树，返回这个结点的父结点
  • 暴力还原二叉树：由当前结点求出整个树的根结点，中序遍历树并存入数组中，查找当前结点并返回下一个

代码

时间：O(N)

空间：O(1)
