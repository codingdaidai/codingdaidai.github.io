Java基本语法 变量 常量

关键字

被赋予特定含义的单词

常见举例：

  • 定义数据类型：class，interface，enum，@interface，byte，short，int，long，char，float等
  • 定义数据类型值：true，false，null
  • 定义流程控制：if，switch，for，continue，while等
  • 定于访问权限修饰符：public，private，protected
  • 定义类，函数，变量修饰符：abstract，final，static，synchronized
  • 定义类与类关系：extends，implements
  • 定义建立实例，引用实例，判断实例：new，this，super，instanceof
  • 用于处理异常：try，finally，catch，throw，throws
  • 用于包：package，import
  • 其他: native，strictfp，transient，volatile，assert

常量

程序执行中固定不变的量

分类：

  • 字面值常量（6类）
    • 字符串 ”HelloWorld“ （双引号）
    • 整数 28，-80
    • 小数 12.34
    • 字符 ‘a’ ‘9’ （单个字符，用单引号）
    • 布尔 true，false
    • 空 null
  • 自定义常量

标识符

给类，方法，变量，常量起名字的字符序列

规则：

  1. 不以数字开头
  2. 不能是关键字
  3. 区分大小写

命名规范：

  1. 类和接口：首字母大写，每个单词首字母大写，如HelloWorld
  2. 变量和方法: 首字母小写，后面开始的单词首字母大写，如getNum
  3. 常量名：所有字母大写，多个单词下划线隔开，如：MAX_VALUE
  4. 包名;全小写，多级用.隔开，域名反写，如cn.itcast.demo
