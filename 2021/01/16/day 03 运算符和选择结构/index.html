java基础 运算符 流程控制-选择结构

算数运算符概述

加 + 减 - 乘 * 除 / 取模(余) % 自增 ++ 自减–

字符和字符串参与加法运算

加法+运算特点：

  • 加号两边是数值型，加法

    ‘a’,‘0’ 等字符型参与时，用该字符对应的数值运算（ASCII值）

  • 加号两边任一边是字符串，进行拼接

自增和自减

  • 单独使用：放在变量前后都一样
  • 参与运算：
    • 变量前：先自增，在运算；
    • 变量后：先以原值进行其他运算，再自增

赋值运算符

  • 基本：=
  • 扩展：+= -= /= *= %= (隐含着强制类型转换)

关系运算符

举例：== != > >= < <=

结果：都是布尔型

逻辑运算符

举例：&&与 ||或 ! 非

结果：布尔型

三元运算符

格式：（关系表达式）?表达式1: 表达式2

含义：关系表达式结果为true，运算结果为表达式1；反之，为表达式2

If语句

3种格式：

  1. if(关系表达式){

    //语句

    }

  2. if (关系表达式){

    //语句体1

    }else{

    //语句体2

    }

  3. if(关系表达式1){

    //语句体1

    }else if(关系表达式2){

    //语句体2

    }else if(关系表达式3){

    //语句体3

    }……

    switch语句

    语句格式;

    switch(表达式){ 表达式可取：byte，short，int，char，string，枚举

    ​ case值1： case后：与表达式比较的值

    ​ 语句体1；

    ​ break；

    ​ case值2：

    ​ 语句体2；

    ​ break；

    ​ ……

    ​ default： default：都不匹配的情况

    ​ 语句体；

    ​ break；

    }
