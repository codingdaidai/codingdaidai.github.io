读取步骤

  • 创建字符流读取文件对象：Reader reader = new FileReader(“路径”);
  • 调用方法读取数据：int data = reader.read(); 读取一个子覅，返回该字符代表的整数；若到达流的末尾，返回-1
  • 异常处理：try…catch或throws
  • 关闭资源：reader.close();

步骤

  • 创建字符流读取文件对象

  • 异常处理
  • 用while循环读写数据
  • 关闭资源

分类

  • BufferedReader

    ​ 构造方法：public BufferedReader(Reader reader)

    ​ 成员方法：public String readLine()：一次读取一行并返回读取到的内容，读不到返回null

  • BufferedWriter

    ​ 构造方法：public BufferedWriter(Reader writer)

    ​ 成员方法：public void newLine()：根据当前操作系统给出对应的换行符

特点

字符缓冲流自带有缓冲区，大小16kB（底层是按照字符数组读写的）

字符缓冲流-一次读写一行

注意：拷贝纯文本文件用字符流；其他如图片，音频，视频文件用字节流

    • 成员方法：public int read() 一次读取一个字节，并返回读到 的内容，读不到返回-1
