HEXO+Gitpages搭建一个博客

​ 2012-1-15

前言

​ 计划搭建一个博客，记录一下自己的学习历程，锻炼自己输出知识和想法的能力。选用了使用较多的Hexo框架和gitpages来部署，虽然过程很简单，还是出了不少问题，简单记录一下过程和遇到的问题。

准备环境

  • win10 64
  • node.js
  • Git
  • Github

以上安装注册都比较简单，网上也分别有很多教程（其实懒得写，以后有时间再来补上）
