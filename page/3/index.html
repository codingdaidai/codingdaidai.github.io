题目

JZ37

统计一个数字在升序数组中出现的次数。

思路

  • 直接遍历：继续向后，同时增加计数变量
  • 二分查找：对于这种有序的数组，常用二分法查找以节省时间
    • 时间复杂度O(nlogn)
    • 二分法找出任何一个等于k的数组元素，记录下他的index
    • 根据这个index向前查找所有等于k的元素数量
    • 根据这个index向后查找所有等于k的元素数量

代码

直接遍历

时间：O(N)

空间：O(1)

二分;

时间：O(NlogN)

空间：O(1)

题目

JZ55

给一个链表，若其中包含环，请找出该链表的环的入口结点，否则，输出null。

思路

  • 集合：遍历整个链表，把结点的地址存入集合，如果有环，链表尾指向的地址会和前面环入口处的地址重复，重复的即为环的入口
  • 双指针：快慢指针都从头开始遍历，快指针每次前进两步，慢指针每次前进一步，如果有环，慢指针会在环内与快指针相遇；相遇后快指针回到链表起点，同时每次前进一步，快慢指针最终在环入口相遇
    • 分析：假设环长度为n，进入环之前结点个数为x,slow在环内走了k个结点,fast绕环走了m圈,则有2(x+k)=x+mn+k 可以得出x = mn - k。此时slow距入口结点还剩 n-k个结点,x=(m−1)n+n−k，即一个指针从链表头节点走到环入口的长度等于另一个指针从相遇的位置走 m-1圈后再走n-k的长度，也就是说两个指针相遇后，让一个指针回到头节点，另一个指针不动，然后他们同时往前每次走一步，当他们相遇时，相遇的节点即为环入口节点。
    • 不容易想到，要写最好记一下

题目

JZ25

复杂链表的复制

输入一个复杂链表（每个节点中有节点值，以及两个指针，一个指向下一个节点，另一个特殊指针random指向一个随机节点），请对此链表进行深拷贝，并返回拷贝后的头结点。

不会 看不懂题目

思路

分析：难点在于这个结点的random指针

如果是普通链表的复制，遍历它，每轮建立新结点，并使前驱结点pre和当前结点cur构建指向关系即可；

现在多了random指针，不能用简单的前后关系来在新表中建立random的指向

  • 哈希表：利用哈希表的查询特点，考虑构建 原链表节点 和 新链表对应节点 的键值对映射关系，再遍历构建新链表各节点的 next 和 random 引用指向即可

  • 拼接和拆分链表：考虑构建 原节点 1 -> 新节点 1 -> 原节点 2 -> 新节点 2 -> …… 的拼接链表，如此便可在访问原节点的 random 指向节点的同时找到新对应新节点的 random 指向节点。

链接：https://leetcode-cn.com/problems/fu-za-lian-biao-de-fu-zhi-lcof/solution/jian-zhi-offer-35-fu-za-lian-biao-de-fu-zhi-ha-xi-/

算法流程：
1.复制各节点，构建拼接链表:

设原链表为 node1→node2→⋯ ，构建的拼接链表如下所示：

node1→node1 new →node2→node2 new →⋯

2.构建新链表各节点的 random 指向：

当访问原节点 cur 的随机指向节点 cur.random 时，对应新节点 cur.next 的随机指向节点为 cur.random.next 。
3.拆分原 / 新链表：

设置 pre / cur 分别指向原 / 新链表头节点，遍历执行 pre.next = pre.next.next 和 cur.next = cur.next.next 将两链表拆分开。
4.返回新链表的头节点 res 即可。

代码

哈希表

时间：O(n) 遍历两次链表

空间：O（n） 使用哈希表存储映射

拼接和拆分:

时间：O(n) 3次遍历

空间：O(1) 创建的新结点实际就是需要的结果

JZ3

输入一个链表，按链表从尾到头的顺序返回一个ArrayList。

注：打印操作通常不去改变原始数据结构。这是与反转链表的区别

思路

  • 栈：遍历链表并入栈，出栈元素写入数组
  • 递归：
    • 函数的功能：反转链表并以ArrayList输出
    • 递归思路：一直递归到链表中只有最后一个元素，然后添加该层递归的链表结点的值到list中，这样最终结果就实现了链表元素的反转

代码

时间：O（n） 共遍历两次所有元素

空间：O（n） 需要大小为n的栈空间，和n的ArrayList

优化：如果用ArrayList的add(index,value)方法，可以指定 index 位置插入 value 值

时间：O（n） 共遍历1次所有元素

空间：O（n） 需要n的ArrayList

时间：O（n）

空间：O(1) 递归栈的空间算不算呢？
